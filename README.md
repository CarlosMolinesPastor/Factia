# <center>A Factia flet app</center>

## Create a environment
mkdir Factia
cd Factia
python3 -m venv .venv
source .venv/bin/activate
pip install flet

## Clone the repository:
git clone https://github.com/CarlosMolinesPastor/Factia.git

## To run the app:
flet run Factia
```
